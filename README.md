# Agent Skills

A collection of skills for AI coding agents following the [Agent Skills](https://agentskills.io/) format.

## Available Skills

### git-commit-guidelines

Best practices for writing clear, consistent git commit messages.

**Use when:**
- Writing commit messages
- Reviewing pull requests for commit quality
- Setting up commit conventions for a new project

**Rules covered:**
- Message structure (subject + body)
- Subject line formatting (imperative mood, 72 chars)
- Body content (what and why, not how)
- Commit scope (one logical change per commit)

## Installation

```bash
npx skills add roin-orca/agent-skills
```
